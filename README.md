# Bash Scripts

Useful Bash scripts for Linux system administration.

## Scripts

### disk_usage.sh
Display disk usage and memory usage.

### service_check.sh
Check SSH service status and system resources.

### server_info.sh
Display hostname, IP address, disk usage and memory usage.

### log_cleanup.sh
Delete log files older than 7 days.

- Linux
- Bash
